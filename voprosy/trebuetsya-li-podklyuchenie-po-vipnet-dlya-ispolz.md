title: Требуется ли подключение по ViPNet для использования Flow?
---

Нет, подключение по ViPNet не является обязательным требованием для работы во Flow.

Для безопасного доступа к системе используется альтернативный защищенный канал связи с российским шифрованием.

Доступ реализован по схеме **ГОСТ TLS.** Это обеспечивает шифрование передаваемых данных между вашим компьютером и сервером Flow в соответствии с российскими стандартами.

Чтобы воспользоваться защищенным каналом, выполните следующие шаги:

1. Установите необходимое программное обеспечение:

   -  Яндекс.Браузер (бесплатный)

   -  CryptoPro CSP (бесплатный)

2. Выполните вход через защищенный канал:

   -  Запустите Яндекс.Браузер.

   -  Перейдите по специальному адресу: <https://flow-gost.migrant-exam.ru/>

   -  Авторизуйтесь в системе, используя свою учетную запись.

После установки программа может показывать уведомления о том, что бесплатная лицензия действует только 90 дней. Это не означает, что соединение перестанет работать.

-  По истечении 90 дней отключаются только функции, связанные с электронной подписью (ЭП).

-  Функция установки защищенного ГОСТ TLS-соединения остается полностью бесплатной и работает неограниченное время.
