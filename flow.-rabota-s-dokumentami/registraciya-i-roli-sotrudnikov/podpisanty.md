Подписант добавляется из карточки в организации со вкладки «Данные подписантов» по кнопке «Добавить подписанта». 

![](./podpisanty.png){width=1452px height=221px}

Далее заполнить все необходимые поля и отметить, какие документы и на каком основании может подписывать подписант.

![](./podpisanty-2.png){width=974px height=375px}

Схема подстановки подписантов:

1. Проверяется есть ли именно в этот день подписант, у которого указан конкретный тип документа. Например, Заявление. Проверяется именно из периода действия полномочий. Если сегодня такой есть - подставляется он в сгенерированный документ. Если такого нет в этот день - проверка идет дальше.

2. Проверяется есть ли в этот день подписант, у которого указан тип документа «Любые документы". Если сегодня такой есть - подставляется он в сгенерированный документ. Если такого нет в этот день - проверка идет дальше.

3. Проверяется есть ли в этот день основной подписант. Если сегодня такой есть - подставляется он в сгенерированный документ. Если такого нет - ничего не генерируется. 

:::info 

**Основной подписант** попадает в генерацию в последнюю очередь.

:::
